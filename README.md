```

  <html>

   </head>

    <p>This is a paragraph.</p>

  </body>
</html>
```

> ``<br> digunakan apabila ingin melakukan baris baru.``
> ``<p> paragraph/paragraf digunakan apabila ingin melakukan penggantian paragraf baru.``
